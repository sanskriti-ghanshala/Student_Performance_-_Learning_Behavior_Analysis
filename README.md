# 📊 Student Performance & Learning Behavior Analysis
Developed a Student Performance &amp; Learning Behavior Analysis dashboard using Python and Power BI. Performed data cleaning, exploratory data analysis, and interactive visualization to analyze the impact of study habits, attendance, sleep, and learning behavior on academic performance.



## 📌 Overview
The **Student Performance & Learning Behavior Analysis** project is an interactive data analytics dashboard developed using **Python** and **Power BI**. The project analyzes various factors such as study habits, attendance, sleep patterns, assignment completion, and learning behavior to understand their impact on students' academic performance.

The dashboard provides meaningful insights that help educators and institutions identify learning trends, monitor student performance, and support data-driven decision-making.

## 📈 Key Insights

- Consistent study habits positively influence exam scores.
- Adequate sleep is associated with improved learning outcomes.

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/student-performance-learning-behavior-analysis.git
```

2. Navigate to the project directory

```bash
cd student-performance-learning-behavior-analysis
```

3. Install the required Python libraries

```bash
pip install -r requirements.txt
```

4. Open the Power BI dashboard (`.pbix`) using Power BI Desktop.
